## 3. 핵심 기능
이 서비스의 핵심 기능은 장바구니 및 게시판, 회원 관리입니다.  
사용자는 관리자가 등록한 상품들을 구매, 장바구니 및 결제를 할 수 있으며
관리자는 상품 관리, 회원이 구매한 상품확인, 회원관리를 할 수 있습니다.
